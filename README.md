Project Description
This project is a personal portfolio website developed as part of the College CCA Individual Assignment.
The website presents academic information, technical skills, projects, and a contact section.

The website is built using only HTML, CSS, and JavaScript without using any frameworks, following academic and course guidelines.

Features
- Responsive design for mobile, tablet, and desktop
- Sticky navigation bar
- Light and Dark mode toggle
- Smooth scrolling navigation
- Skills section with progress bars
- Project showcase section
- Resume download feature
- Contact form integrated using Formspree
- Deployed using GitHub Pages

Technologies Used
HTML5
CSS3 (Flexbox, Grid, CSS Variables)
JavaScript
